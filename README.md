# RAG-Based Customer Support Assistant

This project is a Retrieval-Augmented Generation (RAG) based system built using LangGraph and HITL.

## Features
- PDF knowledge base processing
- Semantic search using embeddings
- Response generation using LLaMA 3.1 (ChatGroq)
- Workflow using LangGraph
- Human-in-the-Loop (HITL) escalation

## Tech Stack
- Python
- LangGraph
- ChromaDB
- ChatGroq

## How to Run
1. Install dependencies
2. Run ingest.py
3. Run main.py
